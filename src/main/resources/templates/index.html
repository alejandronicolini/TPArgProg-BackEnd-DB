EndPoints para la Base de datos SQL

Atencion: excepto los metodos Get, el resto tienen impletentado JWT (Bearer Token)

Post: /auth/login

"nombreUsuario": "xxxxx xxxx",
"password": "xxxxxx"

Los endpoint son:

Tabla Persona

Get: /personas/listado

Post: /personas/new (no implementado en angular)

Put: /personas/update/id (el id es 1, ya que hay una sola persona

Delete: /personas/delete/id (no implementado en angular)

Get: /personas/id (consulta por id, el id es 1)

Tabla InfoPersonal

Get: /info_personal/listado (no implementado en angular)

Post: /info_personal/new (no implementado en angular)

Put: /info_personal/update/id

Delete: /info_personal/delete/id (no implementado en angular)

Get: /info_personal/id (consulta por id, el id es 1)
